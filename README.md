## Mensaje clave

Epsylon es una plataforma de entrenamiento profesional enfocada en entrevistas tecnicas y busqueda laboral asistida por IA.

## Caracteristicas del programa

### 1) Copiloto de entrevista en tiempo real

- Asistencia contextual durante entrevistas tecnicas con sugerencias en vivo.
- Teleprompter discreto para leer respuestas sin perder el flujo de la conversacion.
- Deteccion de plataforma (Zoom, Teams, Meet) para adaptar mejor el contexto.
- Atajos globales para controlar el copiloto sin cambiar de ventana.

### 2) Simulacros de entrevista

- Entrevistas de practica con IA en formato conversacional.
- Configuracion por rol y nivel para entrenar escenarios especificos.
- Feedback final con puntos de mejora para reforzar tu preparacion.

### 3) Busqueda de empleo automatizada

- Busqueda en multiples fuentes de empleo desde un solo lugar.
- Filtros por rol, experiencia, ubicacion y modalidad remota.
- Gestion del pipeline de candidaturas con estados de seguimiento.
- Soporte para generar contenido de aplicacion y seguimiento.

### 4) Base de conocimiento personalizada

- Carga y gestion de documentos para usar contexto propio.
- Recuperacion de fragmentos relevantes durante la asistencia.
- Mejora de respuestas de IA con informacion real del usuario.

### 5) Herramientas de IA para entrevistas

- OCR para extraer texto desde capturas o imagenes.
- STT para transcripcion de voz a texto.
- TTS para escuchar sugerencias cuando necesites manos libres.

### 6) Experiencia Desktop enfocada en discrecion

- Interfaz de escritorio pensada para uso operativo durante sesiones reales.
- Ajustes de comportamiento visual y de control para minimizar friccion.
- Personalizacion de preferencias desde un panel de configuracion central.

### 7) Plataforma con capas completas

- Desktop app para operacion diaria.
- Backend para logica, autenticacion y servicios.
- Web para acceso, gestion de cuenta y flujo comercial.
- Billing integrado para planes y control de acceso.
